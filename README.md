# Fatema Tuj Johra Academic Website

Static academic website prepared for GitHub Pages.

## Update Personal Assets

1. Add the CV PDF at `assets/Fatema_Tuj_Johra_CV.pdf`.
2. Add a profile photo at `assets/profile.jpg`.
3. In `index.html`, replace the portrait placeholder with:

```html
<img src="assets/profile.jpg" alt="Fatema Tuj Johra" />
```

4. Replace `fatema.johra@example.com` with the preferred public email address.

## Publish on GitHub Pages

1. Create a GitHub repository.
2. Upload these files to the repository root.
3. In GitHub, open Settings -> Pages.
4. Set the source to deploy from the main branch.
5. Save and wait for the published URL.
